# UI设计规范（C端用户版）

## 1. 设计概览

本UI设计规范专为C端用户优化，采用现代化、友好、富有活力的设计语言，注重视觉吸引力和用户体验。设计风格年轻、时尚、易用，适合个人用户和创作者使用。在保持品牌一致性的同时，通过更丰富的色彩、更柔和的圆角、更流畅的动画，打造愉悦的使用体验。

## 4. 布局系统

### 4.1 间距系统（8px基础单位，C端更宽松）

- **基础单位**：8px
- **间距层级**（C端更宽松的间距）：
  - XS: 4px（紧密元素）
  - SM: 8px（小间距）
  - MD: 16px（中等间距）
  - LG: 24px（大间距）
  - XL: 32px（超大间距）
  - XXL: 48px（最大间距）
  - XXXL: 64px（页面区块间距）

### 4.2 容器与边距（C端更宽松）

- **页面最大宽度**：1440px（C端更聚焦的内容宽度，超宽屏居中显示）
- **内容边距**（更宽松的边距）：
  - 移动端：20px
  - 平板：32px
  - 桌面端：40px
  - 大屏：48px
- **组件间距**（更舒适的间距）：
  - 卡片间距：24px（移动端）/ 32px（桌面端）
  - 区块间距：48px（移动端）/ 64px（桌面端）
  - 元素间距：12px（移动端）/ 16px（桌面端）

### 4.3 网格系统
- **采用12列网格系统**
- **栅格间距**：24px（桌面端），16px（移动端）
- **响应式断点**：
  - 移动端：< 768px
  - 平板：768px - 1024px
  - 桌面端：1024px - 1440px
  - 大屏：> 1440px

### 4.4 布局框架（C端优化）

- **顶部导航栏**：固定高度 72px（C端更高，更易点击），固定在顶部
- **左侧边栏**：可折叠，默认宽度 280px（C端更宽，更易操作），折叠后 72px
- **内容区域**：自适应宽度，最小宽度 360px（移动端友好）
- **右侧属性栏**：固定宽度 320px（编辑页面）

- **阴影**（更柔和的阴影）：

- **圆角**：12px（标准）/ 16px（大卡片）（C端更大的圆角）
- **过渡**：所有属性 300ms cubic-bezier(0.4, 0, 0.2, 1)（更流畅的动画）

## 13. 设计原则

1. **简洁性**：保持界面简洁，去除不必要的装饰，突出核心功能
2. **一致性**：在整个产品中保持统一的设计语言和交互模式
3. **效率性**：优化操作流程，减少用户操作步骤
4. **可读性**：确保文本和数据易于阅读和理解
5. **响应式**：完美适配不同屏幕尺寸和设备
6. **可访问性**：确保所有用户都能方便使用
7. **现代感**：采用最新的设计趋势，保持产品竞争力
